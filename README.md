## ğŸ¤– Agent Descriptions

| Agent | Purpose | Tools | Model |
|-------|---------|-------|-------|
| **Supervisor** | Orchestrates workflow, creates execution plans | - | Reasoning LLM |
| **Data Explorer** | Schema discovery, data profiling, quality checks | pandas_tools, sql_tools | Fast LLM |
| **SQL Agent** | Text-to-SQL generation, data querying | sql_tools | Fast LLM |

> *Analytics Agent uses reasoning model for complex queries (why, predict, recommend)
